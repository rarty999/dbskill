## 用户主诉

做抖音油画风读书号(账号刚起步、无粉丝、内容积累期),需要每天能稳定产出读书长视频文案。希望确定一个「写作契约」,下次新窗口只输入书名就能直接产出 3500 字稿子,不再反复沟通。

## 已得出的结论

- 账号定位:抖音油画风书单号
- 形式:AI 配音 + 油画轮播,不出镜
- 时长:12-14 分钟,字数 3500 字(±200)
- 钩子默认 A 痛点画面型;变体 B 反常识结论型 / C 金句切入型
- 七层结构:钩子(7%) + 作者背书(8%) + 5 个观点段(75%) + 结尾收束(10%)
- 观点段六件套:概念命名 + 案例/比喻 + 关键转折句 + 作者原话「」+ 反问句 + 金句压尾
- 节奏五律:单句成段 / 核心句切两半 / 反问做分隔 / 对比结构 / 金句压尾
- 金句三公式:对比型(A 才是 X,B 不过是 Y)/ 反差型(你以为 X,其实是 Y)/ 绝对型(孩子不是 X 长大的,是 Y 长大的)
- 比喻是命脉:每篇必含 2-3 个独立具象比喻
- 禁忌:排比堆叠、emoji、危险词(真相 / 无所不能 / 如释重负 等 AI 味形容词)
- **育儿/敏感赛道关键词避雷(2026-06-06 加入)**:涉及育儿暴力 / 未成年人冲突词必须替换,否则 0 推荐(《去情绪化管教》实测仅 5 播放,推荐流量 0%)

## 用户已否决的方向

- 3500 字文章版 —— 否决理由:文章稿和口播稿是两套语言系统,文章稿一字不能用,且 AI 味重、八股化、无认知落差
- 5000 字版本 —— 否决理由:目前驾驭不了这个长度
- 真人出镜 —— 否决理由:本账号定位就是不出镜的油画风书单号

## 待验证假设

- 油画书单号 2026 年是否仍有推流红利(用户判断"今年还是有很多做成功的案例",未独立验证)
- 钩子 A 痛点画面型是否适配所有书(哲学/认知类书可能更适合 B 或 C)
- 3500 字是否是该账号的最优长度(未来可能需要测试 5000 字或更短版本)
- 「先接后教」等 5 大观点抽象方法,是否能稳定从任意一本书里提取出来

## 推荐下一步

下次产出读书文案直接走 dbs-content,调用格式 `/dbs-content 书名:《XX》 按上次契约出`。除非该书不适配 A 钩子、或需要不同字数/重点观点,否则不再反问,直接按契约产出。前 5-10 本书产出后,回看完播率和转化数据,如发现某类书需要其他钩子或长度,再调整契约。

**调用规则**:出口播稿前自检一遍,凡是涉及育儿 / 未成年人 / 亲密关系冲突的书,口播稿里以上高危词出现总次数 ≤ 3 次。超过就改写。

**适用范围**:抖音平台。其他平台(B 站 / 视频号 / 西瓜)审核标准不同,可视情况放宽。

**同行参考稿子常用比喻库**(书单号文案硬通货,可持续积累):
- 种树 / 浇花 / 篮子 / 飞机氧气面罩 / 井里的人 / 镜子

**调用接口(用户下次调用时的精简格式)**:
```
/dbs-content
书名:《XXX》
按上次契约出
```

可选附加参数(默认即按契约,以下情况才补):
- 「钩子用 B/C 版」——书不适合痛点切入
- 「字数 X 字」——需要更短/更长
- 「重点讲 XX」——已读过书,知道哪个观点最该被讲

**核心诉求提醒**:用户要的是稳定产出能力,不是单本质量的极致。契约价值 > 单稿价值。
